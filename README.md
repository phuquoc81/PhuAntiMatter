# PhuAntimatter

Minimal PhuAntimatter game scaffold with:

- a browser-based reactor UI in `/frontend`
- a lightweight Node server in `/backend`
- in-memory economy, auth, leaderboard, and payment method handlers
- database and token scaffolding in `/database` and `/tokens`

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Test

```bash
npm test
```
